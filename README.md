# ReyhanRestaurant

This is the front-end of a restaurant website created using HTML, SASS, CSS, and JavaScript. The purpose of this project is to showcase my skills in front-end development and design.

## Technologies Used

- HTML
- SASS
- CSS
- JavaScript

## Features

- Responsive design for different screen sizes
- Navigation menu with smooth scrolling to sections
- Interactive menu with hover effects and animations
- Contact form with validation using JavaScript

## Getting Started

To view this project, simply clone the repository and open the index.html file in your browser.

https://github.com/Nargeskhl/ReyhanRestaurant.git
